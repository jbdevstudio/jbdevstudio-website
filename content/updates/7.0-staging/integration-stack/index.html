Red Hat JBoss Developer Studio 7.x - Integration Stack Updates

Build 7.0.0.Beta5

How Do I Install?


  1. Start JBDS. Go to Help > Install New Software... > Add...

  2. Paste in the URL of the update site at left. Browse the site for updates you'd like to install. Restart when prompted.

This site contains the latest stable milestone updates for Red Hat JBoss Developer Studio.

For more information, see http://www.redhat.com/products/jbossenterprisemiddleware/developer-studio/.

If you do not already have a copy of Red Hat JBoss Developer Studio 7.x, click here to download it free.

Point your Red Hat JBoss Developer Studio at the following site:

  • Red Hat JBoss Developer Studio 7.x - Integration Stack :: For use with jBPM, Drools, JBoss ESB or JBoss WebServices
    • https://devstudio.jboss.com/updates/7.0-staging/integration-stack/
      • Also available for offline installation via zip:
      • jbdevstudio-integration-stack-updatesite-7.0.0.Beta5.zip
