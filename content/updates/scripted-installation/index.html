Red Hat JBoss Developer Studio - Scripted Installation

Scripted Installation

There are two major ways to install Red Hat JBoss Developer Studio:

a) Using the izpack installer jar, which will install a self-contained product which includes Eclipse:

  • Download the jar, then run java -jar /path/to/jbdevstudio-*.jar
b) Using an update site to install into an existing Eclipse you've already installed:

  • Using the Eclipse Marketplace: https://marketplace.eclipse.org/content/red-hat-jboss-developer-studio-kepler, OR
  • Using the JBoss Developer Studio update site: https://devstudio.jboss.com/updates/7.0/

In both cases, this can be done interactively, or can be scripted for headless or hands-free installation (eg., should you want to install JBoss Developer Studio onto multiple computers in parallel).

  • To script the izpack installer to create installation(s) headlessly, see: Headless Installer
  • To script installation into an existing Eclipse install, optionally also installing other features from JBoss Central or another site, see: Bring Your Own Eclipse
