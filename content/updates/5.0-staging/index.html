Red Hat JBoss Developer Studio 5.x - Updates

How Do I Install?


  1. Start JBDS. Go to Help > Check for updates...

  2. Select which updates you'd like to install. Restart when prompted.

This site contains the latest released updates for Red Hat JBoss Developer Studio. For more information, see
http://www.redhat.com/products/jbossenterprisemiddleware/developer-studio/.

If you do not already have a copy of Red Hat JBoss Developer Studio, click here to purchase or download it free.

Point your Red Hat JBoss Developer Studio at one or more of the following sites:

  • Red Hat JBoss Developer Studio 5.x - Core :: Includes updates from JBoss, Eclipse.org & other vendors
    • https://devstudio.jboss.com/updates/5.0/

  • Red Hat JBoss Developer Studio 5.x - SOA Tooling :: For use with jBPM, Drools, JBoss ESB or JBoss WebServices
    • https://devstudio.jboss.com/updates/5.0/soa-tooling/
      • Also available for offline installation via zip:
      • jbdevstudio-soa-tooling-updatesite-5.0.5.v20130802-1009-H5-GA.zip
  • Red Hat JBoss Developer Studio 5.x - Extras :: Third-Party Extensions & Tools
    • https://devstudio.jboss.com/updates/5.0/extras/

  • Red Hat JBoss Developer Studio 5.x - Technology Preview :: Additional Extensions & Tools
    • https://devstudio.jboss.com/updates/5.0/techpreview/

Note: If you install Red Hat JBoss Developer Studio updates into an existing Eclipse 3.7.2 (Indigo SR2) installation, you will most likely need to tweak your eclipse.ini file to give Eclipse more memory than is allocated by default. This should work for you but is currently Unsupported. Please see the following tips:
