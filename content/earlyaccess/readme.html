Early Access - Read Me


JBoss is seeking participants for an early access program for the Red Hat JBoss Developer Studio 7.0 release. This will offer users a deeper understanding of the new release. Through their participation, users will have the opportunity to directly leverage expertise from JBoss engineering to develop their applications using the bleeding edge features included in JBoss Developer Studio 7.0. At the same time, it is an opportunity for the JBoss Developer Studio product team to gather feedback and input from the participants.

Thank you for expressing your interest in participating in this program. This document will provide you with all the information you need to get started.

Program Structure and Terms

The JBoss Developer Studio Early Access program will run as a unmanaged program in which any number of users may choose to participate. The program will commence in March 2013 and complete by the GA date of the product.

Although these Early Access releases will not be officially supported by Red Hat Global Support team, the JBoss Developer Studio product team will directly assist with participant questions. At the conclusion of the program, we expect customers to use the GA binaries in their ongoing development process. After the GA release, any kind of support for Early Access releases will cease to exist.

Early Access Website

The Early Access program will include binary installer downloads at the above location, and may at some later date also include an update site for incremental updates as JBoss Developer Studio 7.0 progresses towards its GA Release.

  • Website: https://devstudio.jboss.com/earlyaccess/

Downloads are available for Linux, Windows and Mac.
